## Key Visual Improvements

### Edit Profile
1. **Pinned Header**: Always visible Cancel/Save buttons
2. **Banner + Avatar Overlap**: Professional, Twitter-like aesthetic
3. **Section Grouping**: Clear visual hierarchy with section titles
4. **Field Containers**: Subtle emerald background for input areas
5. **Character Counter**: Live feedback for bio length
6. **Large Tap Targets**: All buttons 44x44 minimum
7. **High Contrast**: White text on dark emerald background

### Profile Screen
1. **Clean Location/Portfolio Display**: Icons + single line
2. **Skillset Chips**: Compact, scannable tags instead of verbose list
3. **Achievement Grid**: Professional 3x2 grid with locked/unlocked states

5. **No Duplication**: Single source of truth for skills (Skillsets)
