## Code Example: Complete Integration

```tsx
import { OfflineModeBanner, CompactOfflineBanner } from 'components/offline-mode-banner';
import { SyncStatusIndicator, SyncStatusBadge } from 'components/sync-status-indicator';
import { cachedDataService, CACHE_KEYS } from 'lib/services/cached-data-service';

function MyScreen() {
  const [bounties, setBounties] = useState([]);

  // Load with caching
  const loadBounties = async () => {
    const data = await cachedDataService.fetchWithCache(
      CACHE_KEYS.BOUNTIES_LIST,
      () => bountyService.getAllBounties(),
      { ttl: 15 * 60 * 1000 } // 15 min cache
    );
    setBounties(data);
  };

  return (
    <View>
      {/* Global banner at top */}
      <OfflineModeBanner showDetails={true} />
      
      {/* Header with inline indicator */}
      <View style={styles.header}>
        <Text>My Screen</Text>
        <CompactOfflineBanner />
      </View>
      
      {/* Content */}
      <ScrollView>
        {bounties.map(bounty => (
          <BountyCard key={bounty.id} bounty={bounty} />
        ))}
      </ScrollView>
      
      {/* Footer with status */}
      <View style={styles.footer}>
        <SyncStatusIndicator detailed={true} size="small" />
      </View>
    </View>
  );
}
```

This visual guide shows how the enhanced offline experience components work together to provide:
- **Clear visibility** of offline status
- **Real-time sync progress** 
- **Cached data** for offline viewing

The implementation is modular, allowing developers to choose the right components for each screen context.
