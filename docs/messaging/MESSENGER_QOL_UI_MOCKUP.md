All interactive elements have minimum 44x44 touch target:
- Pin header dismiss: 44x44
- Message actions: 60px height
- Long-press area: Entire message bubble

## Theme Consistency

All new components follow the existing Bounty theme:
- **Primary**: Emerald (#059669, #047857, #065f46)
- **Accent**: Amber (#fbbf24) for pins
- **Status**: Blue (#60a5fa) for read
- **Error**: Red (#ef4444) for failed/report
- **Text**: Emerald-200 (#d1fae5) on dark backgrounds
- **Border**: Emerald with transparency

This maintains visual consistency across the entire app.
